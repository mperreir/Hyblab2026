## Instructions de déploiement

Toutes les commandes doivent être exécutées à la racine du projet.

Pour le premier lancement de l'application :

- Installer les dépendances :
```bash
    npm install
```
- Créer la base de données et initialiser les tables :
```bash
node actu/api/setup.js
```
- Lancer l'application :
```bash
npm start
```
- Accéder à l'application :
+ + http://localhost:8080/actu/
