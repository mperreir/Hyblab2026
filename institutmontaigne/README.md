## Instructions de déploiement

Dossier `institutmontaigne/developpement` lancer la commande suivante : 

```bash
cd institutmontaigne/developpement;
npm install;
npm run build;
```

Le projet utilise React et Vite, le build du code dans `institutmontaigne/developpement` se trouve dans le dossier `institutmontaigne/public`.
